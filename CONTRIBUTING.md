# Contributing

Thanks for contributing to AetherLens Media Downloader.

## Development Setup

1. Install Node.js 18+ and npm 9+.
2. Install dependencies: `npm install`
3. Start dev mode: `npm run dev`
4. Run lint: `npm run lint`
5. Validate build: `npm run build`

## Pull Requests

- Keep PRs focused and easy to review.
- Explain what changed and why.
- Include test notes.
- Add screenshots for UI changes.
- Update docs when behavior changes.

## Checklist

- Lint/build pass.
- No secrets or local artifacts committed.
- No unrelated refactors.
- Compatibility considered for Windows, macOS, and Linux.
