How It Works

Simple flow for non-technical users.

STEP 1

Download the installer for your OS and complete the setup wizard.

STEP 2

Paste any supported media URL to fetch metadata and available formats.

STEP 3

Select output format, quality, and destination preferences.

STEP 4

Start downloads and manage progress from the queue screen.

About Installer Security Warnings

Your download has started. On Windows or macOS, you might still see a trust warning because AetherLens is an open-source app and we are not paying for commercial code-signing certificates at this stage.

This is usually a signing/reputation warning, not proof of malware. For clarity, you can always inspect the source code and releases in this repository.
