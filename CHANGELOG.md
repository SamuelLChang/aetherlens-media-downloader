# Changelog

All notable changes to this project are documented in this file.

## [Unreleased]

### Added

- Added a one-time `What's New` modal after version changes to highlight recently shipped improvements.
- Added a reusable release QA checklist at `docs/RELEASE_QA_CHECKLIST.md` for pre-tag validation.

### Changed

- Improved first-run setup behavior: continuing with missing required dependencies now requires explicit limited-mode confirmation.
- Added sidebar update badge support so available updates are visible outside the Info page.
- Enhanced failed-download UX with direct recovery actions: `Retry`, `Copy Details`, and context-aware install guide links.
- Pinned TypeScript to `5.5.4` for compatibility with the current ESLint TypeScript parser stack.

## [0.1.0] - 2026-02-22

### Added

- Initial public metadata and community health files.
- First-run setup wizard and repo cleanup baseline.
