# Security Policy

## Supported Versions

Security fixes are provided on a best-effort basis for the latest release line.

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |
| < 0.1.0 | No        |

## Reporting a Vulnerability

Please do not disclose security issues publicly first.

Report privately with:

- affected version
- reproduction steps
- impact
- optional mitigation suggestion

Maintainers will acknowledge and triage as quickly as possible.
